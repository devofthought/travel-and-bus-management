### Run

You need to start the server first. In the [server](./backend/) directory, execute the following commands:

```
$ yarn
$ yarn dev
```

Then, to spawn clients, go to the [client](./frontend/) directory and execute the following commands:

```
$ yarn
$ yarn dev
```
